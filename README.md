# Open Engine Room Simulator (OERS)

An open-source, web-based engine room simulator inspired by commercial marine engineering training systems.

## Goal

Build a modern simulator for marine engineering education using modern web technologies.

## Project Status

Planning Phase

## Planned Features

- Engine Room Mimic Diagrams
- Emergency Generator
- Diesel Generators
- Main Engine
- Boilers
- Fuel Systems
- Cooling Systems
- Alarm Management
- Instructor Mode
- Scenario Editor
- Trend Graphs
- Training Exercises
